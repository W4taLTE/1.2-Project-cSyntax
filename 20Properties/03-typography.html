üí° Fundamental Properties

CSS Typography mengontrol tampilan teks pada halaman web. Memahami property typography penting untuk desain yang readable dan estetik.

typography.css
/* Font Properties */
body {
  font-family: 'Segoe UI', sans-serif;
  font-size: 1rem;
  font-weight: 400;
  font-style: normal;
}

/* Text Properties */
h1 {
  color: #9333ea;
  text-align: center;
  text-transform: capitalize;
  text-decoration: underline;
}

/* Spacing Properties */
p {
  line-height: 1.6;
  letter-spacing: 0.5px;
  word-spacing: 2px;
}

üìù Penjelasan Detail:

‚Ä¢ font-family - menentukan jenis font
‚Ä¢ font-size - ukuran teks (px, em, rem, %)
‚Ä¢ font-weight - ketebalan font (100-900 atau normal/bold)
‚Ä¢ text-align - perataan teks (left, center, right, justify)
‚Ä¢ line-height - spasi antar baris (penting untuk readability)

üí° Pro Tips

üéØ Best Practices

Gunakan system font stack untuk performance, set base line-height 1.5 untuk readability, dan gunakan clamp() untuk responsive typography.

best-practices.css
/* ‚úÖ Good - System font stack */
body {
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}

/* ‚úÖ Good - Responsive typography */
h1 {
  font-size: clamp(1.75rem, 5vw, 2.5rem);
}

/* ‚ùå Avoid - Fixed pixel values */
p {
  font-size: 16px;
  line-height: 24px;
}

/* ‚ùå Avoid - Too many web fonts */
body {
  font-family: 'Poppins', 'Montserrat', 'Raleway', sans-serif;
}

üìù Penjelasan Detail:

‚Ä¢ System fonts - load lebih cepat dari web fonts
‚Ä¢ clamp() - buat font-size responsive dengan minimum dan maximum
‚Ä¢ Relative units - gunakan rem/em untuk aksesibilitas
‚Ä¢ Limit font variations - 2-3 font family maksimal
‚Ä¢ Vertical rhythm - pertahankan konsistensi spasi vertikal
‚Ä¢ Variable fonts - single file dengan banyak variasi
