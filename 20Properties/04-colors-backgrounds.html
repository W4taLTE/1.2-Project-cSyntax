üí° Fundamental Properties

CSS Colors & Backgrounds mengontrol warna teks, latar belakang, dan efek visual lainnya. Memahami property ini penting untuk desain yang menarik.

colors.css
/* Color Properties */
body {
  color: #333333; /* Text color */
  background-color: #ffffff; /* Background color */
}

/* Background Properties */
.header {
  background-image: url('header-bg.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

/* Gradient Background */
.gradient-box {
  background: linear-gradient(45deg, #9333ea, #a855f7);
}

üìù Penjelasan Detail:

‚Ä¢ color - mengatur warna teks
‚Ä¢ background-color - warna latar belakang
‚Ä¢ background-image - gambar latar belakang (url/gradient)
‚Ä¢ background-repeat - pengulangan gambar (repeat/no-repeat)
‚Ä¢ background-position - posisi gambar (top, center, bottom)

üí° Pro Tips

üéØ Best Practices

Gunakan warna dengan kontras yang baik untuk aksesibilitas, optimasi gambar latar untuk performa, dan manfaatkan CSS variables untuk konsistensi warna.

best-practices.css
/* ‚úÖ Good - CSS Variables for colors */
:root {
  --primary-color: #9333ea;
  --secondary-color: #a855f7;
}

.btn {
  background: var(--primary-color);
}

/* ‚úÖ Good - Accessible color contrast */
.text {
  color: #333;
  background: #fff;
  /* Contrast ratio at least 4.5:1 */
}

/* ‚ùå Avoid - Low contrast colors */
.bad-contrast {
  color: #999;
  background: #eee;
}

/* ‚ùå Avoid - Large uncompressed images */
.hero {
  background-image: url('large-uncompressed-image.png');
}

üìù Penjelasan Detail:

‚Ä¢ CSS Variables - mudah maintenance dan konsistensi warna
‚Ä¢ Color contrast - minimal 4.5:1 untuk teks normal
‚Ä¢ Optimasi gambar - kompres untuk performa loading
‚Ä¢ Gradients over images - lebih ringan dari gambar
‚Ä¢ Fallbacks - selalu sediakan fallback untuk browser lama
‚Ä¢ Prefixed properties - gunakan prefix untuk support lama
