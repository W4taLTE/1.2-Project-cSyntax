üí° Nesting Fundamental

CSS Nesting memungkinkan Anda menulis rules CSS dalam struktur bersarang yang lebih intuitif, mirip dengan HTML.

nesting.css
/* Traditional CSS */
.header {
  background: #9333ea;
}

.header h1 {
  color: white;
}

/* Dengan CSS Nesting */
.header {
  background: #9333ea;

  & h1 {
    color: white;
  }
}

üìù Penjelasan Detail:

‚Ä¢ & - Nesting selector merujuk ke parent selector
‚Ä¢ Bisa nested hingga beberapa level (tapi jangan berlebihan)
‚Ä¢ Mendukung semua jenis selector (class, id, pseudo-class, dll)
‚Ä¢ Lebih mudah dibaca dan di-maintain untuk komponen kompleks

üí° Pro Tips

üéØ Best Practices

Gunakan nesting dengan bijak - maksimal 3-4 level kedalaman. Hindari nesting berlebihan yang membuat CSS sulit dibaca dan di-maintain.

best-practices.css
/* ‚úÖ Good - Moderate nesting */

  border-radius: 8px;

  & .header {
    padding: 1rem;
    background: #9333ea;

    & h2 {
      color: white;
    }
  }
}

/* ‚ùå Avoid - Too deep nesting */

  & .content {
    & .article {
      & .header {
        & h1 {
          color: #9333ea;
        }
      }
    }
  }
}

üìù Penjelasan Detail:

‚Ä¢ Gunakan BEM dengan nesting - kombinasikan manfaat keduanya
‚Ä¢ Hindari over-qualification - jangan nested selector yang sudah spesifik
‚Ä¢ Group related styles - nesting bagus untuk komponen yang terkait
‚Ä¢ Perhatikan output CSS - nesting bisa menghasilkan CSS yang lebih besar
‚Ä¢ Gunakan tools - PostCSS atau Sass jika butuh kompatibilitas luas
