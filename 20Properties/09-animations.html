üí° Animation Fundamentals

CSS Animations memungkinkan Anda membuat animasi kompleks dengan menentukan keyframe dan properti animasi.

animations.css
/* Definisi keyframes */
@keyframes slideIn {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

/* Penerapan animasi */
.box {
  animation: slideIn 0.5s ease-out forwards;
  background: #9333ea;
}

/* Animasi multiple keyframes */
@keyframes bounce {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-20px);
  }
}

üìù Penjelasan Detail:

‚Ä¢ @keyframes - Mendefinisikan tahapan animasi
‚Ä¢ animation-name - Nama keyframe yang akan digunakan
‚Ä¢ animation-duration - Durasi animasi (contoh: 2s)
‚Ä¢ animation-timing-function - Fungsi waktu (ease, linear, etc)
‚Ä¢ animation-iteration-count - Jumlah pengulangan (infinite untuk terus menerus)

animation-values.css
/* Animation Properties */
animation-name   /* Nama keyframe */
animation-duration /* Durasi (1s, 200ms) */
animation-timing-function /* Fungsi waktu */
animation-delay   /* Penundaan */
animation-iteration-count /* Jumlah */
animation-direction /* Arah */
animation-fill-mode /* Fill mode */

/* Timing Functions */
ease     /* Default */
linear   /* Konstan */
ease-in  /* Slow start */
ease-out /* Slow end */
cubic-bezier() /* Kustom */

üìù Penjelasan Detail:

‚Ä¢ Fill modes - Mengontrol state sebelum/sesudah animasi
‚Ä¢ Iteration count - Dapat berupa angka atau 'infinite'
‚Ä¢ Direction - normal, reverse, alternate, alternate-reverse
‚Ä¢ Play state - running atau paused
‚Ä¢ Nilai durasi umum: 300-500ms untuk interaksi, 1-3s untuk animasi panjang

üí° Pro Tips

üéØ Best Practices

Gunakan animasi dengan bijak untuk meningkatkan pengalaman pengguna tanpa mengorbankan performa.

best-practices.css
/* ‚úÖ Good - Performant properties */
@keyframes goodExample {
  to {
    transform: translateX(100px) scale(1.2);
    opacity: 0.8;
  }
}

/* ‚ùå Avoid - Expensive properties */
@keyframes badExample {
  to {
    width: 500px;
    margin-left: 100px;
  }
}

/* ‚úÖ Good - Prefers reduced motion */
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}

üìù Penjelasan Detail:

‚Ä¢ Gunakan transform/opacity - Properti ini paling efisien untuk dianimasikan
‚Ä¢ Hindari animasi layout properties - width, height, margin menyebabkan reflow
‚Ä¢ Gunakan will-change - Untuk memberi tahu browser properti yang akan berubah
‚Ä¢ Hormati prefers-reduced-motion - Beberapa pengguna sensitif terhadap animasi
‚Ä¢ Batasi animasi simultan - Terlalu banyak animasi membebani performa
