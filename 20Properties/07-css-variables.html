üí° Fundamental Variables

CSS Variables (Custom Properties) memungkinkan Anda menyimpan nilai yang dapat digunakan kembali dalam stylesheet Anda, membuat pengelolaan desain lebih konsisten dan mudah di-maintain.

variables.css
/* Deklarasi variabel global */
:root {
  --primary-color: #9333ea;
  --secondary-color: #a855f7;
  --text-color: #333;
  --spacing-unit: 1rem;
}

/* Menggunakan variabel */
.header {
  background: var(--primary-color);
  padding: calc(var(--spacing-unit) * 2);
}

/* Variabel dengan fallback */
.btn {
  background: var(--btn-color, var(--primary-color));
}

üìù Penjelasan Detail:

‚Ä¢ :root - Scope global untuk variabel CSS
‚Ä¢ --variable-name - Sintaks deklarasi variabel CSS
‚Ä¢ var() - Fungsi untuk menggunakan variabel
‚Ä¢ calc() - Perhitungan dengan variabel
‚Ä¢ var(--var, fallback) - Fallback value jika variabel tidak ada

}

üí° Pro Tips

üéØ Best Practices

Gunakan CSS Variables untuk membuat sistem desain yang konsisten, mudah di-maintain, dan dapat dikustomisasi.

best-practices.css
/* ‚úÖ Good - Organized and reusable */
:root {
  /* Colors */
  --color-primary: #9333ea;
  --color-secondary: #a855f7;

  /* Spacing */
  --space-sm: 0.5rem;
  --space-md: 1rem;
}

/* ‚ùå Avoid - Unorganized and hard to maintain */
.header {
  --color1: #9333ea;
  --space1: 10px;
}
.footer {
  --color2: #a855f7;
  --space2: 15px;
}

üìù Penjelasan Detail:

‚Ä¢ Organize by category - Kelompokkan variabel berdasarkan jenis (warna, spacing, dll)
‚Ä¢ Use semantic names - Nama yang deskriptif (--color-primary bukan --color1)
‚Ä¢ Define globally first - Mulai dengan :root untuk variabel global
‚Ä¢ Scope appropriately - Gunakan scope komponen untuk variabel spesifik
‚Ä¢ Document your variables - Tambahkan komentar untuk penjelasan
