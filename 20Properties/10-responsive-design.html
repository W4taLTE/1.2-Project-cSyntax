üí° Media Query Fundamentals

Media Queries memungkinkan Anda menerapkan CSS berbeda berdasarkan karakteristik perangkat seperti lebar layar, orientasi, dan resolusi.

responsive.css
/* Breakpoint dasar */
@media (max-width: 768px) {
  .container {
    padding: 1rem;
  }
}

/* Range lebar layar */
@media (min-width: 576px) and (max-width: 992px) {
  .card {
    width: 50%;
  }
}

/* Orientasi perangkat */
@media (orientation: portrait) {
  .header {
    height: auto;
  }
}

üìù Penjelasan Detail:

‚Ä¢ @media - Aturan untuk menentukan media queries
‚Ä¢ max-width - Maksimum lebar viewport
‚Ä¢ min-width - Minimum lebar viewport
‚Ä¢ orientation - Portrait atau landscape
‚Ä¢ and - Kombinasi multiple conditions

üìù Penjelasan Detail:

‚Ä¢ hover/pointer - Deteksi jenis perangkat input
‚Ä¢ Gunakan unit relatif (em, rem, %) daripada piksel absolut

    }
  }

üìù Penjelasan Detail:

‚Ä¢ Mobile-first - Mulai dari small screens baru ke larger screens
‚Ä¢ Gunakan em/rem - Lebih aksesibel daripada px untuk breakpoints
‚Ä¢ Minimal breakpoints - Gunakan hanya saat diperlukan
‚Ä¢ Content-based breakpoints - Sesuaikan dengan konten bukan perangkat
‚Ä¢ Test berbagai perangkat - Jangan hanya bergantung pada ukuran viewport
