üí° Scroll Snap Fundamental

CSS Scroll Snap memungkinkan Anda membuat scroll yang "snap" ke posisi tertentu, ideal untuk galeri, carousel, dan tampilan halaman penuh.

scroll-snap.css
/* Container Properties */
.scroll-container {
  scroll-snap-type: y mandatory;
  overflow-y: scroll;
  height: 100vh;
}

/* Child Elements */
.scroll-item {
  scroll-snap-align: start;
  height: 100vh;
}

/* Horizontal Example */
.horizontal-scroll {
  scroll-snap-type: x mandatory;
  overflow-x: scroll;
  display: flex;
}

üìù Penjelasan Detail:

‚Ä¢ scroll-snap-type - Container property menentukan perilaku snap
‚Ä¢ scroll-snap-align - Child property menentukan posisi snap
‚Ä¢ scroll-snap-stop - Mengontrol apakah scroll melewati item
‚Ä¢ scroll-margin/scroll-padding - Menyesuaikan area snap

üìù Penjelasan Detail:

‚Ä¢ Axis Options - Kontrol arah scroll (x, y, block, inline)
‚Ä¢ Alignment Points - start, center, end menentukan posisi snap
‚Ä¢ Scroll Behavior - smooth scroll bisa dikombinasikan

‚Ä¢ Hindari overuse - Tidak semua halaman perlu scroll snap
‚Ä¢ Test di berbagai device - Perilaku bisa berbeda di touch vs mouse
